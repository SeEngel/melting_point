# melting_point

## Python environment

This code depends on the 'uv' tool and a Python 3.12 virtual environment.

Prerequisites
- Install uv following the official guide:
    https://docs.astral.sh/uv/getting-started/installation/

- Create a virtual environment named .venv using uv:
    uv venv .venv --python==3.12


Once the environment is active, install any project dependencies and run the module as usual.
